# Zeige einen Wochentag

Schreibe eine Funktion `getWeekDay(date)`, um den Wochentag in Kurzform anzuzeigen: 'MO', 'DI', 'MI', 'DO', 'FR', 'SA', 'SO'.

Zum Beispiel:

```js no-beautify
let date = new Date(2012, 0, 3);  // 3. Jan 2012
alert( getWeekDay(date) );        // sollte "DI" ausgeben
```
